File Version Comparator

Source code

Uses Merkle trees to detect discrepancies in csv files

This project simulates the use of Merkle trees in a business setting that tracks sales reports. Merkle trees are data structures that take advantage of a hierarchical hashing system that allows for instant comparison of files and offers an efficient method to correct updates.

Genomics Analyzer

Source code

Statistical and biological insight into DNA sequences

This program supports .fasta and .prot_table file formats and calculates interesting benchmarks such as the number of base pairs, GC content percentage, longest homopolymer, and the overall coding fraction percentage. Additionally, basic statistics are generated on intergenic regions. Finally, additional gene parameters can be specified to see a gene's base pair and amino acid breakdown.
