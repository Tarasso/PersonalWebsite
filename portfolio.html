File Version Comparator

Source code

Uses Merkle trees to detect discrepancies in csv files

This project simulates the use of Merkle trees in a business setting that tracks sales reports. Merkle trees are data structures that take advantage of a hierarchical hashing system that allows for instant comparison of files and offers an efficient method to correct updates.
